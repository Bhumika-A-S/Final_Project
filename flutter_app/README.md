# TipTrack Flutter Frontend (Starter)

Minimal Flutter frontend that calls the existing TipTrack FastAPI backend.

Quick run (requires Flutter SDK):

```bash
# from this folder
flutter pub get
flutter run -d chrome   # or an emulator/device
```

Notes:
- API base URL defaults to `http://10.0.2.2:8000` (Android emulator) or `http://localhost:8000` for web.
- This is a starter scaffold — expand pages and models as needed.
